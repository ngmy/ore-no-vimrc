mysqli_connect

(PHP 5, PHP 7)

mysqli_connect — mysqli::__construct() のエイリアス

説明

この関数は次の関数のエイリアスです。 mysqli::__construct()

mysqli::__construct() のドキュメントにも mysqli_connect() を使った手続き型の例は掲載されていますが、 ここでも簡単に例を示します。

例

例1 mysqli_connect() の例

<?php
$link = mysqli_connect("127.0.0.1", "my_user", "my_password", "my_db");

if (!$link) {
    echo "Error: Unable to connect to MySQL." . PHP_EOL;
    echo "Debugging errno: " . mysqli_connect_errno() . PHP_EOL;
    echo "Debugging error: " . mysqli_connect_error() . PHP_EOL;
    exit;
}

echo "Success: A proper connection to MySQL was made! The my_db database is great." . PHP_EOL;
echo "Host information: " . mysqli_get_host_info($link) . PHP_EOL;

mysqli_close($link);
?>

上の例の出力は以下となります。

Success: A proper connection to MySQL was made! The my_db database is great.
Host information: localhost via TCP/IP
